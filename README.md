# Media Coverage Analysis of Newspaper Articles (2020–2023)

## Overview

This project visualises counts of newspapers on COVID-19 donations coverage across 4 media outlets between August 2020 and March 2023. The visualisation aggregates the number of included articles per month and visualizes coverage trends over time.

## Objectives

- Quantify article inclusion across newspaper outlets
- Aggregate monthly article counts
- Visualize media coverage trends over time

## Methods

The analysis was conducted using R and the following packages:

- tidyverse
- dplyr
- ggplot2
- lubridate
- readxl

## Repository Structure

data/ – raw and processed datasets  
scripts/ – R scripts used for analysis  
figures/ – generated visualizations  
docs/ – methodological notes

## Visualization

The final visualization shows the monthly number of included articles between August 2020 and March 2023.
